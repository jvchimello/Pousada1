# Pousada1
Pousada Secreta  Projeto de um site institucional para uma pousada fictícia, desenvolvido como parte de um curso de desenvolvimento web.  

## Sobre o projeto
O objetivo deste projeto foi praticar conceitos fundamentais de HTML e CSS, criando uma página simples, responsiva e visualmente organizada para apresentação de uma pousada.

##  Funcionalidades
- Estruturação semântica com HTML5
- Estilização com CSS
- Layout responsivo
- Seções informativas (sobre, quartos, localização e contato)
- Integração com mapa (Google Maps)

##  Tecnologias utilizadas
- HTML5
- CSS3

##  Responsividade
O site foi adaptado para funcionar em diferentes tamanhos de tela, incluindo dispositivos móveis.
